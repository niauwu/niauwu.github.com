錯誤註解

  1. 在 ; 後註解 //

  2. 註解後 // 沒有於列末輸入 ;

    display: table; //我是表格

//我是表格;
display: table;

把以下 table.css 放入 Hexo\themes\landscape\source\css 資料夾內

#css_table {
  display: table;
  border-collapse: collapse;
}
.css_tr {
    display: table-row;
}
.css_td {
    display: table-cell;
    border: 1px solid white;
    text-align: center;
    padding-left: 10px;
    padding-right: 10px;
}

把以下 link 插入 Hexo\themes\landscape\layout\index.ejs 檔案內

<link rel="stylesheet" href="/css/table.css"/>

Excel 實用密技

密技一：
儲存格內換行 ： Alt+Enter

密技二：
第2列 換到 第1列 ： 剪下第2列，在第1列按右鍵>插入減下的儲存格(E)。
快捷鍵法：在第2列按右減再按T，在第1列按右鍵，再按E

密技三：
固定欄位，選擇要度定的欄位的右邊或下面一格，「檢視」(Alt W)→「凍結窗格」(F)→「凍結窗格」(F)。就會固定所選擇的欄位的左邊或上面欄位。

如何製作無限循環背景

本頁上方 niauwu 做的 banner 無限循環背景不錯吧!
讓我們來 DIY 做無限循環背景吧！

挑選圖片、編輯圖片

  1. 選擇一張要循環的圖片，例如一張寬 100px 的圖片。
  2. 用修圖軟體把圖片寬度改成三倍，例如新增一個 300px的圖片檔，之後把圖片按照順序排列，第一張 x=0，第二張 x=100，第三張 x=200，之後把第二張水平反轉。存檔之後就變成可以無限循環的圖片了。

Hexo

如何發布文章

新建文章

    hexo new "articleTitle"

打完指令後就可以在 /source/_posts/ 發現多一個articleTitle.md

編輯文章

1.用Notepad++，開啟articleTitle.md，修改編碼(編碼>編譯成 UTF-8 碼(檔首無 BOM)
2.修改標題、分類、標籤

title: title文章標題 #標題可以改成中文的，如"新文章"
date: 2014-06-16 10:00:00 #發表日期，一般不更動
categories: categories分類 #文章分類
tags: [tag1標籤1，tag2標籤2] #文章標籤，多於一項用此格式

3.於---後開始編輯文章，要使用markdown格式
4.編輯完後存檔

產生文章

hexo g  //hexo generate

在本地架站&查看文章(可選、可自行選擇要不要)

hexo s //hexo server 在本地架站

在本地查看文章 http://localhost:4000
如果要關閉本地架站，請服用(Control+C)

可直接在本地編輯、存檔(Control+S)，在瀏覽器重新整理(F5)，就可以看到更新的文章，因為Git bash會自動更新(update)。
這樣就不用→編輯→存檔→產生→在本地架站&查看文章→關閉本地架站→
可以直接→編輯→存檔→重新整理瀏覽器→
但是要上傳之前要先產生文章’hexo g’喔

部署(上傳)文章

hexo d  //hexo deploy，deploy(佈署、發布、部署)
輸入github帳號
輸入github密碼

一行代碼產生文章+發布文章

hexo d -g //hexo deoply -generate

Translator Toolkit

如何製作Google翻譯詞彙

  1. 於Google雲端硬碟，新增Google試算表
  2. 重新命名標題為我的詞彙
  3. 於第一列(row)輸入en zh-TW pos description
    • en(英文)
    • zh-TW(繁體中文)、zh-CN(簡體中文)
      • 詳細的語言環境請參考語言環境
    • pos(詞性)
      • noun(名詞)
      • verb(動詞)
      • adjective(形容詞)
      • adverb(副詞)
      • pronoun(代名詞)
      • preposition(介系詞)
      • conjunction(連接詞)
      • interjection(感嘆詞)
    • description(詳細說明)
  4. 可以把第一列(row)凍結，這樣用滑鼠滾輪上下滾動檢視詞彙時，仍然可以知道這一欄位要填入什麼
    • 檢視>凍結列>凍結 1 列
  5. 使用驗證資料限定pos(詞性)填入的資料，設定完後就可以直接下拉選單選擇該詞的詞性
    • 選取D行(column)>資料>驗證>條件:改成項目清單>輸入noun,verb,adjective,adverb,pronoun,preposition,conjunction,interjection
    • 資料無效的處理方式：>選取拒絕輸入內容
    • 外觀:>勾選顯示說明>輸入填入詞性
  6. 第二列(row)以後，就可以填入詞彙

什麼，你要多人協作編輯此詞彙？
那就共用吧～
有兩種共用方式
第一種知道共享網址的任何人都可以編輯
1.點右上角共用>變更>知道連結的使用者>存取權:任何人(無須登入)選擇可編輯
第二種受到邀請的人才可以編輯
1.點右上角共用>邀請別人:輸入被邀請者的名稱或電子郵件地址
