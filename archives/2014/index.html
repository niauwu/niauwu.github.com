手動刪除 TWG 多媒體教學廣播系統

讓老師無法操控或監視我們的電腦螢幕 ，通常廣播系統在學生端是不會有[關閉]的選項，不過換個想法把廣播系統刪掉就好了，趕快近來看看如何刪除吧!

一般而言，教學廣播系統是無法從學生端關閉的，不過換個想法，軟體要執行就要有執行檔而執行檔剛好就在學生端這裡，所以乾脆就把執行檔刪了吧，要找到執行檔其實還蠻簡單的，通常在裝軟體時的執行檔與資料都會放在C槽下的Program Files的資料夾下，所以就去找找看吧，以TWG教學廣播系統來看，TWG是安裝在C:\Program Files\TWG\eClassroom裡面，而其中最重要的執行檔就是Rsagent.exe，但是如果直接刪出的話會出現，檔案使用中無法刪除之類的警告，所以必須借助工作管理員的幫忙，工作管理員大家應該都知道吧，所以吧工作管理員叫出來(Ctrl+Alt+Del)，在處理程序下面找到Rsagent.exe按結束處理程序，然後刪掉就可以了，不過TWG有以個保護機制就是當被工作管理員結束處理程序時會自動再啟動一次，所以刪除的時間非常短，但是可以用一 個方法，就是先在執行檔Rsagent.exe的上面按右鍵刪除但先不要，按是
，接著到處理程序Rsagent.exe的地方，按結束處理程序，也先不要按是，然後把兩個警告的視窗重疊，用非常迅速的速度，連按兩個是，就完成拉。

Appserv 免安裝

appserv免安裝原來這麼簡單喔
AppServ免安裝有可能嗎?有可能，看以下教學
首先當然要有AppServ沒有的去下載吧
正常AppServ的目錄下有Apache2.2 and MySQL and php5 and www and Uninstall-AppServ2.5.10.exe 免安裝其實只是把Apache2.2和MySQL啟動而已啦 所以來教大家如何啟動
<第一步>:叫出cmd來，win+R輸入cmd，正常會出現C:Documents and Settings\使用者名稱>
<第二步>:切換到AppServ的根目錄，假設是在D槽，輸入D:再輸入cd D:\appserv\apache2.2\bin再輸入httpd.exe -k -install就會啟動apache。
但是如果出現httpd.exe: Syntax error on line 56 of D:/AppServ/Apache2.2/conf/httpd.conf: ServerRoot must be a valid directory不是做錯了，是httpd.conf裡面的安裝地點和當按放的地方不一樣。
解決方法用記事本打開httpd.conf(檔案在apache2.2\conf裡面)，打開後Ctrl+H收尋目標輸入一開始AppServ裝的地方，假設一開始AppServ是裝在C:\AppServ，就要輸入C:/，取代為書入目前AppServ資料夾的地方，本例是在D:\AppServ，所以要輸入D:/，按全部取代，關掉記事本，是，回到cmd按鍵盤上鍵確定，如果沒反應就代表成功了，開啟流覽器在網址列輸入http://localhost/有出現網頁就成功了。
但是cmd.exe這個程式關掉的話apache也會跟著關掉喔，所以cmd.exe不能關掉喔，當然你要把apache關掉就直接把cmd關掉就可以瞜。 至於開啟mysql的方法我還要再研究一下

MP3壓縮測試

MP3壓縮測試 比較MP3 File Size(Bitrate) Reduce Software與DietMP3的壓縮功力。

測試時用到軟體:

  1. MP3 File Size(Bitrate) Reduce Software 壓縮MP3用
  2. DietMP3 壓縮MP3用
  3. GoldWave 試聽壓縮完的MP3

步驟:

  1. 用MP3 File Size(Bitrate) Reduce Software，依照該軟體中可以壓縮的選項，把原始MP3檔分別壓縮成各個BitRate位元速率，其中可壓縮的選項為Reduce to x kbps，x=320,256,224,192,160,144,128,128,112,96,80,64,56,48,40,32,24,16,8。總共有19種選擇。

  2. 用GoldWave個別試聽壓縮完的MP3並與原始MP3檔做比較。測試完的結果如下圖

    小評論:從上圖中可發現Reduce to 64~112 kbps都是不錯的選擇。

  3. 用DietMP3，壓縮原始MP3檔，依照該軟體中可以壓縮的選項，其中可壓縮的選項，有三個可選擇，分別是player播放器、Quality品質、effect效果，其中player播放器有五個選項可選(按照第一個英文字母升冪排列)分別是cell phone手機、MP3 potable可攜式MP3、Other MP3 Device其他MP3裝置、Palm PDA掌上型PDA、Pocket PC口袋型電腦，再來Quality品質方面有四個選項可選(按照品質高低排)分別是CD、FM Radio廣播、tape錄音帶、Telephone手機，最後effect效果有兩的選項可選分別是Stereo立體聲、Mono單聲 如下如所示，所以總共有5×4×2=40種選擇

  4. 用GoldWave個別試聽壓縮完的MP3並與原始MP3檔做比較。測試完的結果如下圖


    小評論:從上圖中可發現檔案的大小和BitRate位元速度有關，且與所選的Quality品質有關，但與所選的Player播放器和effect效果無關。由此可知DietMP3雖然選項較多，但是其中player和effect都與檔案無關，所以選擇時只要在一所要的Quality品質即可，另外值得一提的是，當Quality選擇Telephone手機的時候，不管我們所選的effect是Stereo立體聲或是Mono單聲道都會被壓縮成Mono單聲。

最後總結:
MP3 File Size(Bitrate) Reduce Software和DietMP3這兩個壓縮MP3的軟體不能說孰優孰劣，因為兩者重疊的部分，剛好就是我們理想且可接受的壓縮範圍，若使用MP3 File Size(Bitrate) Reduce Software時就可以選擇 Reduce to 64~112 kbps壓縮率達0.5~0.88，若使用DietMP3壓縮時就可以選擇Quality=CD、FM Radio，player和effect可以不用選(其實effect選stereo比較好)，其實CD=96BitRate、FM Radio=64BitRate。我有發現用DietMP3的Quality=FM Radio會比同等級的、檔案同樣大小的MP3 File Size(Bitrate) Reduce Software 的Reduce to 64 kbps還要好一點(要很仔細聽才聽得出來)。

問題 TSSTCORP CDDVDW TS-L633A 無法讀取空白CD-R

TSSTCORP CDDVDW TS-L633A 無法讀取空白CD-R
為甚麼我的光碟機無法讀取空白CD-R
光碟機型號：TSSTCORP CDDVDW TS-L633A 三星 薄型
系統：Win7
筆電型號：Asus N81vp

狀況：某天某人叫我幫忙燒音樂CD，由於mp3檔案在隨身碟裡面，所以用燒錄軟體製作音樂CD，但是狀況來的，我的光碟機竟然無法讀空白的CD，what’s the matter?

我試過了
可以讀空白DVD，可以讀有資料的DVD，可以讀有資料的CD-R，但就是讀不到空白的CD-R，而且也試過兩個不同廠牌的CD-R，而且確定CD-R是可以燒錄的，因為我拿去學校燒是可以燒的，而且用DVDInfoPro軟體也知道可以讀寫DVD，也可以讀寫CD-R。但是就是無法寫入CD-R。

上網查了之後發現該型號的光碟機不少人有這種問題

X有人說去裝置管理員把光碟機停用，再啟用就可以了，但是此法不適合我的電腦
http://tw.group.knowledge.yahoo.com/pc-shop/article/view?aid=5

X有人說去裝置管理員卸載光驅驅動，不行的話IDE控制器也卸載，但是此法不適合我
http://www.5f800.com/thread.php?t=244

X有人跟我遇到一樣的問題，下面的回覆說，薄型的本來就很差，直接買新的比較快
http://news.cdbest.net/html/26/t-262426.html

X有人說是win7系統問題 1.輸入services.msc 2.把shell hardware detection服務停止並禁止啟動，但是此法也不適合我
http://zhidao.baidu.com/question/238421483.html

但是其實第一次把CD-R放進去時有出現Win7的視窗說是要燒錄資料還是要燒入CD/DVD播放器可以播的光碟，我想說我要用燒錄軟體來燒，所以就把視窗關掉，沒想到之後都不能讀空白CD-R了，是怎樣？難道跟系統會有關係嗎？或許換成XP就可以了嗎？還是要重灌？
但是由於電腦是和我弟共用所以也無法擅自決定要不要重灌，之後再跟他討論看看。

後來想說原本的外接DVD燒錄機壞了，筆電的燒錄機又讀不到CD-R光碟，所以乾脆買一台外接是的DVD燒錄器好了，所以上網選了一台，mobile01網友推薦的外接式DVD燒錄器，BUFFALO品牌的外接DVD燒錄機，Buffalo DVSM-PC58U2V/B。
http://www.mobile01.com/topicdetail.php?f=493&t=2267054&p=1

所以就去 PChome 訂了一台，看起來還不錯，過幾天拿到商品再跟大家分享。

把以下 table.css 放入 Hexo\themes\landscape\source\css 資料夾內

#css_table {
  display: table;
  border-collapse: collapse;
}
.css_tr {
    display: table-row;
}
.css_td {
    display: table-cell;
    border: 1px solid white;
    text-align: center;
    padding-left: 10px;
    padding-right: 10px;

把以下 link 插入 Hexo\themes\landscape\layout\index.ejs 檔案內

<link rel="stylesheet" href="/css/table.css"/>

如何製作無限循環背景

本頁上方 niauwu 做的 banner 無限循環背景不錯吧!
讓我們來 DIY 做無限循環背景吧！

挑選圖片、編輯圖片

  1. 選擇一張要循環的圖片，例如一張寬 100px 的圖片。
  2. 用修圖軟體把圖片寬度改成三倍，例如新增一個 300px的圖片檔，之後把圖片按照順序排列，第一張 x=0，第二張 x=100，第三張 x=200，之後把第二張水平反轉。存檔之後就變成可以無限循環的圖片了。

上CSS

#infiniteBG {    
    width: 100%;
    height: 80px;
    background-image: url("https://niauwu.github.com/img/infiniteBackground.jpg");
    background-position: 0px 0px;
    background-repeat: repeat-x;

    -webkit-animation: bgMove 10s linear infinite;
    -moz-animation: bgMove 10s linear infinite;
    -ms-animation: bgMove 10s linear infinite;
    -o-animation: bgMove 10s linear infinite;
    animation: bgMove 10s linear infinite;  
}

@-webkit-keyframes bgMove {
    from {background-position:0% 0%;}
    to {background-position: 300px 0%;}
}
@-moz-keyframes bgMove {
    from {background-position:0% 0%;}
    to {background-position: 300px 0%;}
}
@-ms-keyframes bgMove {
    from {background-position:0% 0%;}
    to {background-position: 300px 0%;}
}
@keyframes bgMove {
    from {background-position:0% 0%;}
    to {background-position: 300px 0%;}
}
