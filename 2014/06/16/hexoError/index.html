Hexo Error 怎麼辦？

狀況說明
要在Hexo使用Disqus時
修改_config.yml
改成如下

disqus_shortname:shortname

出現以下錯誤

$ hexo g
[error] { rawMessage: 'Unable to parse.',
  parseLine:79,
  snippet: '該列的文字'
  parseFile: null,
  message: 'Config file load failed',
  name: 'HexoError',
  stack: undefined }

你看出來是為甚麼了嗎？
反白看答案:後少一個空白
所以正確答案是

disqus_shortname: shortname

我可是上網找資料找好久，突然才想到
忍不住想說一句，報錯不好好報，至少寫一下”你少一個空白”。
